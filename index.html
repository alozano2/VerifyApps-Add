Política de Privacidad

Última actualización: 2026

La aplicación El Impostor ha sido desarrollada como una aplicación gratuita. Esta aplicación se proporciona tal cual y está destinada a ser utilizada tal como está.

Recopilación y uso de la información

La aplicación no recopila directamente datos personales de los usuarios.

Sin embargo, la aplicación utiliza servicios de terceros que pueden recopilar información utilizada para identificarle.

Publicidad

La aplicación utiliza Google AdMob para mostrar anuncios. Google AdMob puede recopilar y utilizar datos como identificadores de dispositivo, dirección IP aproximada y datos de interacción con anuncios para mostrar publicidad personalizada o no personalizada.

Para más información sobre cómo Google utiliza los datos, puede consultar:

https://policies.google.com/privacy

Datos recopilados por terceros

  • Identificadores de dispositivo
  • Datos de uso relacionados con anuncios

Seguridad

No recopilamos datos personales directamente. Los servicios de terceros utilizados implementan sus propias medidas de seguridad.

Cambios en esta política

Podemos actualizar esta Política de Privacidad ocasionalmente. Se recomienda revisar esta página periódicamente para cualquier cambio.

Contacto

Si tiene alguna pregunta sobre esta Política de Privacidad, puede contactar a través del correo indicado en la ficha de Google Play.
